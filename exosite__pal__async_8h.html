Setup PAL for IO.

Parameters

Any HW or SW initialization should be performed in here

This function is meant to perform any one time initialization and/or setup. This will be called everytime exosite_start is called. It is expected that this will do a DNS lookup of #host.

When this is complete, it must call exoPal_ops_s::on_start_complete(). Either as the last thing this function does, or sometime later from async activity.
